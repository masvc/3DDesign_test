# 3Dインタラクティブポートフォリオ

Three.jsを使用した3Dポートフォリオサイトのプロトタイプです。

## 機能
- 3D空間での作品展示
- マウスによるカメラ操作
- 作品との基本的なインタラクション

## 技術スタック
- Three.js
- Vite
- JavaScript

## 開発方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## プロジェクト構造
```
src/
  ├── main.js      # エントリーポイント
  └── style.css    # スタイル定義
```
